### MR-001: Offline purchase is the current alternative

- **Type**: Problem Evidence
- **Signal / Evidence**: User stated that fans currently buy tickets at the stadium and the demo should let users buy through an app.
- **Implication for product**: v1 must prioritize match browsing, seat selection, checkout, and e-ticket retrieval over advanced marketing or membership features.
- **Confidence**: Medium
- **Source / Asset**: User discovery batch, 2026-06-08.

### MR-002: Demo scope favors manual payment over gateway integration

- **Type**: Positioning Hypothesis
- **Signal / Evidence**: User explicitly rejected payment gateway integration and requested a configured transfer QR plus admin confirmation.
- **Implication for product**: v1 checkout must support manual transfer, pending confirmation, rejection, expiry, and ticket issuance after admin confirmation.
- **Confidence**: High
- **Source / Asset**: User discovery batch, 2026-06-08.
